
# ASRA AI — Phishing Detection System

## 🚀 Overview
ASRA AI is a machine learning-based system designed to detect phishing URLs using engineered features and classification models.

## 🧠 Features
- URL-based feature extraction
- Real-time phishing detection via API
- Lightweight backend for fast inference
- Frontend interface for user interaction

## ⚙️ Tech Stack
- Python
- Machine Learning
- FastAPI / Backend API
- React (Frontend)

## 🏗️ Architecture
User Input → Feature Extraction → ML Model → Prediction → Output
